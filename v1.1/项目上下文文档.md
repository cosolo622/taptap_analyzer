# TapTap舆情分析器 v1.1 - 项目上下文文档

## 一、项目概述

### 1.1 项目目标
- 爬取TapTap平台鹅鸭杀手游评价数据
- 进行情感分析、主题分类、一句话总结
- 生成可视化图表和舆情报告
- 支持实时监控和历史数据分析

### 1.2 技术栈
- **后端**: Python FastAPI + PostgreSQL
- **前端**: Vue 3 + Element Plus + ECharts
- **爬虫**: Playwright (异步)
- **NLP**: jieba分词 + AI分析

## 四、API接口

### 4.1 数据接口

| `/api/crawler/status` | GET | 获取爬虫状态 |
| `/api/crawler/incremental` | POST | 增量更新（新增） |
