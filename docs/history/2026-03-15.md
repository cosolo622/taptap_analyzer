### 2. 星级显示问题修复

**问题**：前端星级显示为空

**原因**：后端代码把星级转成了字符串 `"X星"`，前端期望数字

**解决方案**：
```python
# 修改前
'星级': f"{r.rating}星" if r.rating else None

# 修改后
'星级': r.rating if r.rating else None
```

**文件**：`v1.1/backend/main.py` 第227行
